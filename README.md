# README #

Ranked API - A Backend platform for providing services to the Ranked Mobile apps.

### What is this repository for? ###

* Ranked API
* 0.1
* Django / Python

### How do I get set up? ###

* Dependencies:  Python 3.5

* Installing:
    - run pip(3) -r requirements.txt
    - run manage.py migrate to setup the DB.
